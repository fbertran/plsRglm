# Dichotomization

This function takes a real value and converts it to 1 if it is positive
and else to 0.

## Usage

``` r
dicho(val)
```

## Arguments

- val:

  A real value

## Value

0 or 1.

## See also

[`ifelse`](https://rdrr.io/r/base/ifelse.html)
