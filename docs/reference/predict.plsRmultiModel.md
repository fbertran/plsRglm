# Predict method for experimental plsRmulti models

Prediction method for `"plsRmultiModel"` objects.

## Usage

``` r
# S3 method for class 'plsRmultiModel'
predict(
  object,
  newdata,
  comps = object$computed_nt,
  type = c("response", "scores"),
  verbose = TRUE,
  ...
)
```

## Arguments

- object:

  An object of class `"plsRmultiModel"`.

- newdata:

  Optional predictor matrix or data frame. For formula-fitted models, a
  data frame with the predictor variables used at fit time.

- comps:

  Number of extracted components to use.

- type:

  Either `"response"` or `"scores"`.

- verbose:

  Should informational messages be displayed?

- ...:

  Not used.

## Value

If `type = "response"`, a matrix of predicted responses. If
`type = "scores"`, a matrix of latent score coordinates.

## See also

[`plsRmulti`](https://fbertran.github.io/plsRglm/reference/plsRmulti.md)
